

# 🌍 Global Pollution Analysis and Energy Recovery (ML Project)

## 📌 Overview

This project focuses on analyzing global pollution data using machine learning techniques to classify countries into pollution severity levels—**Low, Medium, and High**. It also explores the relationship between pollution and energy consumption to generate meaningful environmental insights and recommendations.

---

## 🎯 Objective

* Classify countries based on pollution severity levels.
* Analyze the impact of pollution on energy consumption and recovery.
* Compare multiple machine learning models to identify the best-performing approach.
* Provide actionable insights for environmental sustainability.

---

## 🛠️ Technologies Used

* Python
* Jupyter Notebook
* NumPy
* Pandas
* Matplotlib / Seaborn
* Scikit-learn

---

## 📂 Dataset

* **Global_Pollution_Analysis.csv**
* Contains data on:

  * CO₂ emissions
  * Industrial waste
  * Air, water, and soil pollution indices
  * Energy consumption
  * Country and year

---

## ⚙️ Project Workflow

### 🔹 1. Data Preprocessing

* Data cleaning (handling missing values & outliers)
* Feature scaling and normalization
* Encoding categorical variables
* Feature engineering:

  * Energy consumption per capita
  * Pollution trends over time

---

### 🔹 2. Model Implementation

#### ✅ Naive Bayes

* Multinomial Naive Bayes for classification
* Fast and efficient for probabilistic predictions

#### ✅ K-Nearest Neighbors (KNN)

* Distance-based classification
* Hyperparameter tuning for optimal K value

#### ✅ Decision Tree

* Rule-based classification model
* Tuned using parameters like max depth and split criteria

---

### 📊 3. Model Evaluation

Each model is evaluated using:

* Accuracy
* Precision
* Recall
* F1-Score
* Confusion Matrix

---

### 📈 4. Visualization

* Confusion matrices
* Classification reports
* Graphs to understand pollution trends and patterns

---
